FAQ

Questions, answered

Do I need API keys to use GenNal?

No. GenNal drives the official CLIs you've already installed and logged into — Codex, Claude, and Gemini. There are no extra API keys and no extra billing on our side.

Does my code or prompts go through your servers?

No. Everything runs locally. GenNal is a cockpit around the CLIs on your own machine; your prompts and code go straight to whichever model CLI you're using.

Can I add a model that isn't listed?

Yes. Any command-line tool works. Register a name, command, and args — ollama, aider, or a local script — and it gets its own live pane like the built-ins.

What platforms are supported?

Windows 10/11, macOS 11+ (Apple Silicon and Intel builds), and Linux as an AppImage. All 64-bit.

Is it free?

Yes, GenNal is free to download and use.
